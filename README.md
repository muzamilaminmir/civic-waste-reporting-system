# CivicFix | Garbage Collection Miss Reporter

CivicFix is a modern, transparent, and public accountability system designed to help citizens report missed garbage collections. By making reports public and tracking resolution times, CivicFix ensures municipal waste management remains efficient and accountable.


## 🚀 Features

- **Interactive Incident Map**: Pinpoint the exact location of missed pickups on a live map.
- **Modern Dashboard**: Real-time analytics showing total reports, active issues, and efficiency metrics.
- **Dark Mode Support**: Seamlessly switch between light and dark themes for better accessibility.
- **Public Accountability**: Every incident is listed publicly to ensure transparency.
- **Dynamic Animations**: Premium UI with staggered row entries, counter animations, and glassmorphism effects.
- **Responsive Design**: Built with Tailwind CSS for a flawless experience across all devices.

## 🛠️ Tech Stack

### Frontend
- **HTML5 & Vanilla JavaScript**: Core logic and structure.
- **Tailwind CSS**: Modern, utility-first styling.
- **Leaflet.js**: Interactive maps and incident markers.
- **Google Fonts (Outfit)**: Premium typography.

### Backend
- **FastAPI (Python)**: High-performance API framework.
- **SQLAlchemy**: SQL toolkit and Object-Relational Mapper.
- **SQLite**: Reliable local database.

## 📦 Getting Started

### Prerequisites
- Python 3.8+
- Modern Web Browser

### Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/muzamilaminmir/civic-waste-reporting-system.git
   cd civic-waste-reporting-system
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   pip install -r requirements.txt
   python main.py
   ```
   *The backend will run on `http://localhost:8000`.*

3. **Frontend Setup**:
   Simply open `frontend/index.html` in your favorite web browser.
